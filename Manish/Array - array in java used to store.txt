Array - array in java used to store homogeneous(same type of data) value together
	- known for fixed size
	- array follow indexing
	- can store null(unknown value)
	- it can even store duplicate
syntax1 :
Datatype[] variable = new Datatype[size of array]

syntax2 :
Datatype variable[] = new Datatype[size of array]

example:
String [] name = new String[3];
String  name[] = new String[3]
name[0] = "Manish"
name[1] = "Jaywant"
name[2] = "Bhagat"
homogeneous(same type of data)(only int, only string, only float)
